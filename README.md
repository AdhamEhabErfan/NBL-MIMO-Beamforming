# Neural Codebook Learning for MIMO Beam Management

## Overview

This repository implements Neural Codebook Learning (NBL)
for 5G/6G beam management using deep learning and hybrid beamforming.

## Features

- CNN-based beamspace learning
- SSB and CSI-RS optimization
- TensorFlow/Keras implementation
- Sionna ray tracing integration
- Performance evaluation
